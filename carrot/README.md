# PARD 6기 웹파트 3차 과제

Tailwind CSS로 **당근마켓 랜딩 페이지 / 메인 페이지**를 구현하였습니다.  
라우팅을 적용하여 검색창의 주황색 돋보기를 클릭하면 메인페이지로 이동하는 기능을 구현하였습니다.
