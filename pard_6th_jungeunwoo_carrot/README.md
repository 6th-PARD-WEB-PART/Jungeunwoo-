# PARD 6기 웹파트 5차 과제

Zustand를 활용하여 랜딩페이지의 검색창에 검색어 입력 후 버튼 클릭 시 입력되 내용이 메인페이지에 보여지도록 구현

데스크탑,테블릿,모바일버전에 따라 메인페이지에 반응형 레이아웃을 적용
